Programming Languages

C++ Code Template 
		 	
				#include <bits/stdc++.h>
				using namespace std;
				
				int main () {
					// solution comes here
				}
This code can be compiled using: 
		  
			  g++ -std=c++11 -O2 -Wall test.cpp -o test
or simply: 
		  
			  g++ test.cpp -o test

Input and Output

cin can be used for input. 
		  
			  int a, b;
			  string s;
			  cin >> a >> b >> s;
cout can be used for output. 
		  
			  int a = 123, b = 456;
			  string s = "monkey";
			  cout << a << " " << b << " " << s << "\n";
The output of this code is: 
		  
			  123 456 monkey
> ***Note:*** \n is used to create a newline and it is faster than endl. However, `endl` flushes the stream while `\n` does not. You can use the following commands to make input and output faster: 
		  
			  ios::sync_with_stdio(0);
			  cin.tie(0); cout.tie(0);
